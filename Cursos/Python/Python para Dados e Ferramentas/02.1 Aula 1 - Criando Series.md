# ðŸ“š ConteÃºdo Explicado

### 1. O que Ã© `pd.Series()` â€” visÃ£o geral
- **DefiniÃ§Ã£o simples:** um array 1-D com rÃ³tulos (Ã­ndice).
- **Estrutura:** `Index` (rÃ³tulos) + `values` (os dados).
- **Assinatura bÃ¡sica:** `pd.Series(data=None, index=None, dtype=None, name=None, copy=False)`  
  - `data`: lista, ndarray, dict, scalar, etc.  
  - `index`: rÃ³tulos; se omitido, Pandas cria `RangeIndex(0, 1, 2...)`.  
  - `dtype`: forÃ§a tipo (ex: `int`, `float`, `str`).  
  - `name`: nome da Series (Ãºtil ao convertÃª-la em coluna de DataFrame).

### 2. Como criar â€” exemplos e explicaÃ§Ãµes
#### a) A partir de uma lista

**SaÃ­da esperada**

> Cada linha mostra: Ã­ndice `0,1,2` â†’ valor. `dtype` detectado automaticamente.

#### b) A partir de uma lista com Ã­ndice e nome
```python

```
**SaÃ­da**


#### c) A partir de um dicionÃ¡rio (chaves viram Ã­ndices)
```python

```

#### d) A partir de numpy


#### e) Scalar + Ã­ndice
```python

```

### 3. Acesso e seleÃ§Ã£o
- `s[0]` ou `s['a']`
- `s.loc['a']` â€” acesso por rÃ³tulo.  
- `s.iloc[0]` â€” acesso por posiÃ§Ã£o inteira.  
- Fatias: `s[1:3]` ou `s['a':'c']`.
