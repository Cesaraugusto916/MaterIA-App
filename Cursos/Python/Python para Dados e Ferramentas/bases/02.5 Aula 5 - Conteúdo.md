## üìç Introdu√ß√£o

Em um DataFrame, saber acessar os dados corretamente √© essencial para qualquer manipula√ß√£o ou an√°lise. A indexa√ß√£o permite selecionar linhas, colunas ou c√©lulas espec√≠ficas, seja por r√≥tulo (nome) ou por posi√ß√£o (n√∫mero). Nesta aula, vamos aprender a usar os m√©todos `loc[]` e `iloc[]`, al√©m de suas vers√µes otimizadas `at[]` e `iat[]`.

**Por que isso √© importante?** Porque quase toda opera√ß√£o em Pandas come√ßa com uma sele√ß√£o: filtrar uma coluna, pegar uma linha espec√≠fica, acessar um valor para modificar. Saber usar esses m√©todos evita erros, melhora a legibilidade do c√≥digo e permite trabalhar com dados de forma precisa.

## üìö Conte√∫do Explicado

### 1. `loc[]` ‚Äî acesso por r√≥tulo

- **O que faz**: Seleciona dados usando os nomes (√≠ndices e colunas).
- **Sintaxe**: `df.loc[linha, coluna]`
- **Exemplos**:
  ```python
  df.loc['seg']              # linha com √≠ndice 'seg'
  df.loc['seg', 'vendas']    # c√©lula espec√≠fica
  df.loc[:, 'vendas']        # todas as linhas da coluna 'vendas'
  df.loc[['seg', 'ter'], ['vendas', 'loja']]  # m√∫ltiplas linhas e colunas
  ```

### 2. `iloc[]` ‚Äî acesso por posi√ß√£o

- **O que faz**: Seleciona dados usando n√∫meros (posi√ß√£o das linhas e colunas).
- **Sintaxe**: `df.iloc[linha, coluna]`
- **Exemplos**:
  ```python
  df.iloc[0]                 # primeira linha
  df.iloc[0, 1]              # c√©lula na linha 0, coluna 1
  df.iloc[:, 2]              # terceira coluna
  df.iloc[1:4, 0:2]          # fatia de linhas e colunas
  ```

### 3. `at[]` e `iat[]` ‚Äî acesso r√°pido a uma c√©lula

- **`at[]`**: por r√≥tulo
  ```python
  df.at['seg', 'vendas']
  ```
- **`iat[]`**: por posi√ß√£o
  ```python
  df.iat[0, 1]
  ```

## üìà Exemplos Pr√°ticos

```python
import pandas as pd

df = pd.DataFrame({
    'dia': ['seg', 'ter', 'qua', 'qui', 'sex'],
    'vendas': [250, 300, 280, 410, 380],
    'loja': ['A', 'A', 'B', 'B', 'A']
})
df = df.set_index('dia')

# Acesso por r√≥tulo
print(df.loc['qua'])

# Acesso por posi√ß√£o
print(df.iloc[2])

# Acesso a c√©lula espec√≠fica
print(df.loc['sex', 'vendas'])
print(df.iloc[4, 0])
```

## üìé Dicas e Armadilhas

- `loc[]` exige que o √≠ndice esteja corretamente definido.
- `iloc[]` pode gerar erro se o n√∫mero estiver fora do intervalo.
- `at[]` e `iat[]` s√£o mais r√°pidos, mas s√≥ funcionam para uma c√©lula.
- Evite misturar `loc[]` com `iloc[]` ‚Äî s√£o l√≥gicas diferentes.
- Sempre verifique o √≠ndice com `df.index` antes de usar `loc[]`.
