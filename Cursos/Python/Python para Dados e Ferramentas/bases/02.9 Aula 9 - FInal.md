# üìç Matrizes: A Tabela que Sabe Calcular

Uma matriz √©, essencialmente, uma tabela de n√∫meros com linhas e colunas. No Numpy, ela √© representada por um array de duas dimens√µes (um "array de arrays"). Enquanto voc√™ usou o Pandas para manipular tabelas de dados, o Numpy √© a ferramenta para fazer **c√°lculos matem√°ticos** com essas tabelas.

A √°lgebra linear, que √© o estudo das matrizes e vetores, √© a base matem√°tica de quase tudo em ci√™ncia de dados:

  - **Machine Learning**: Modelos de regress√£o, redes neurais e sistemas de recomenda√ß√£o usam multiplica√ß√£o de matrizes para aprender com os dados.
  - **Estat√≠stica**: C√°lculos de correla√ß√£o e covari√¢ncia s√£o opera√ß√µes matriciais.
  - **Gr√°ficos 3D**: Jogos e renderiza√ß√µes gr√°ficas usam matrizes para rotacionar e posicionar objetos no espa√ßo.

Aprender a manipular matrizes n√£o √© um passo opcional. √â o que diferencia um manipulador de dados de um verdadeiro analista.

# üìö Conte√∫do Explicado: As Ferramentas da Matriz

### 1\. Criando Matrizes com `np.array()` e Conhecendo seu Formato

Para criar uma matriz, basta passar uma "lista de listas" para a fun√ß√£o `np.array()`.

```python
# Criamos uma matriz 2x2: duas linhas, duas colunas
matriz = np.array([[1, 2], [3, 4]])
```

Para saber o tamanho exato da sua matriz, use o atributo **`.shape`**. Ele retorna uma tupla com as dimens√µes: `(linhas, colunas)`. Sempre verifique o `.shape` antes de qualquer opera√ß√£o complexa para evitar erros.

-----

### 2\. `np.transpose()`: Invertendo a Perspectiva

A transposi√ß√£o √© uma opera√ß√£o simples, mas crucial. Ela transforma as linhas de uma matriz em colunas e as colunas em linhas.

  - **`np.transpose(matriz)`**: Pega a matriz e "vira" ela. O formato original `(m, n)` se torna `(n, m)`.
  - A analogia mais simples √© a de uma planilha: a primeira linha se torna a primeira coluna, a segunda linha se torna a segunda coluna, e assim por diante.

-----

### 3\. `np.dot()` e `np.matmul()`: Multiplicando o Poder

A multiplica√ß√£o de matrizes √© diferente da multiplica√ß√£o comum de n√∫meros. Ela segue uma regra fundamental: o n√∫mero de colunas da primeira matriz deve ser igual ao n√∫mero de linhas da segunda. Se essa regra n√£o for respeitada, o Numpy retornar√° um erro.

  - **`np.dot(A, B)`** e **`np.matmul(A, B)`**: Ambas as fun√ß√µes realizam a multiplica√ß√£o de matrizes. Para arrays 2D, elas funcionam de forma id√™ntica.
  - **Importante**: A multiplica√ß√£o de matrizes **n√£o √© comutativa**, ou seja, `np.dot(A, B)` n√£o √© o mesmo que `np.dot(B, A)`.

-----

## üìà Exemplos Pr√°ticos: M√£o na Massa

Vamos criar duas matrizes e ver as opera√ß√µes em a√ß√£o.

```python
import numpy as np

# Criando as matrizes
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

print("--- Matriz A original ---")
print(A)
print(f"Formato da Matriz A: {A.shape}\n")

# Transposi√ß√£o
A_transposta = np.transpose(A)
print("--- Matriz A Transposta ---")
print(A_transposta)
print(f"Formato da Matriz A Transposta: {A_transposta.shape}\n")

# Multiplica√ß√£o de matrizes
multiplicacao_AB = np.dot(A, B)
print("--- Multiplica√ß√£o A x B ---")
print(multiplicacao_AB)
```

**Percebeu a diferen√ßa?** A multiplica√ß√£o resultou em uma nova matriz, onde cada elemento √© o produto da soma das linhas da primeira matriz pelas colunas da segunda. √â uma opera√ß√£o fundamental para muitos modelos matem√°ticos.

## üß™ Demonstra√ß√£o Aplicada: Resolvendo um Problema Real

Imagine um professor que tem as notas de 3 alunos em 2 disciplinas e quer calcular a m√©dia ponderada, onde cada disciplina tem um peso diferente.

```python
# Notas dos 3 alunos (linhas) em 2 disciplinas (colunas)
notas = np.array([[8, 7],
                  [9, 6],
                  [7, 10]])

# Pesos de cada disciplina
# Coluna 0: peso 0.6 | Coluna 1: peso 0.4
pesos = np.array([[0.6],
                  [0.4]])

print("--- Matriz de Notas ---")
print(notas)
print(f"Formato das notas: {notas.shape}\n")

print("--- Vetor de Pesos ---")
print(pesos)
print(f"Formato dos pesos: {pesos.shape}\n")

# Multiplica√ß√£o para m√©dia ponderada
# Numpy faz o c√°lculo de forma eficiente, multiplicando cada nota pelo seu peso
media_ponderada = np.dot(notas, pesos)

print("--- M√©dia Ponderada por Aluno ---")
print(media_ponderada)
```

**O que aconteceu?**
O Numpy realizou a multiplica√ß√£o de matrizes para n√≥s. O resultado √© um novo array, onde cada valor representa a m√©dia ponderada de cada aluno. Por exemplo, para o primeiro aluno, o c√°lculo foi `(8 * 0.6) + (7 * 0.4)`, resultando em `7.6`. Esse √© o poder da √°lgebra linear\!

-----

## üìé Dicas e Armadilhas: O que o Profissional Faz

  - **Verifique as dimens√µes**: Antes de qualquer multiplica√ß√£o, use `.shape` para garantir que o n√∫mero de colunas da primeira matriz √© igual ao n√∫mero de linhas da segunda.
  - **`@` como atalho**: Em vers√µes mais recentes do Python, o operador `@` √© um atalho para a multiplica√ß√£o de matrizes: `A @ B`.
  - **Transposi√ß√£o para Alinhar**: Se a multiplica√ß√£o der erro de dimens√£o, a transposi√ß√£o √© a primeira coisa a se tentar para alinhar as matrizes.
