## üìç Introdu√ß√£o

Filtrar dados significa selecionar apenas os registros que atendem a uma determinada condi√ß√£o. √â uma das tarefas mais comuns em an√°lise de dados: encontrar clientes com mais de 30 anos, vendas acima de R$ 500, ou produtos fora de estoque. Nesta aula, vamos aprender como aplicar condi√ß√µes l√≥gicas para extrair subconjuntos de um DataFrame usando m√°scaras booleanas.

**Por que isso √© importante?** Porque a filtragem √© a base para an√°lises direcionadas, relat√≥rios personalizados e visualiza√ß√µes espec√≠ficas. Saber aplicar condi√ß√µes corretamente permite responder perguntas reais com precis√£o e efici√™ncia.

## üìö Conte√∫do Explicado

### 1. Filtragem simples com condi√ß√£o

```python
df[df['coluna'] > valor]
```

- Retorna apenas as linhas onde a condi√ß√£o √© verdadeira.

### 2. Filtragem com `loc[]`

```python
df.loc[df['coluna'] == 'valor']
```

- Permite combinar filtragem com sele√ß√£o de colunas.

### 3. Operadores l√≥gicos

- `&` ‚Äî E
- `|` ‚Äî OU
- `~` ‚Äî N√ÉO

```python
df[(df['idade'] > 30) & (df['cidade'] == 'SP')]
```

> Par√™nteses s√£o obrigat√≥rios em express√µes compostas.

### 4. `.isin()` ‚Äî m√∫ltiplos valores

```python
df[df['categoria'].isin(['A', 'B'])]
```

### 5. `.between()` ‚Äî intervalo de valores

```python
df[df['preco'].between(10, 50)]
```

## üìà Exemplos Pr√°ticos

```python
import pandas as pd

df = pd.DataFrame({
    'produto': ['A', 'B', 'C', 'D'],
    'preco': [10, 25, 50, 80],
    'estoque': [100, 0, 200, 50]
})

# Produtos com pre√ßo acima de 30
print(df[df['preco'] > 30])

# Produtos fora de estoque
print(df[df['estoque'] == 0])

# Produtos com pre√ßo entre 20 e 60
print(df[df['preco'].between(20, 60)])
```

## üß™ Demonstra√ß√£o Aplicada

Imagine que voc√™ quer analisar apenas as vendas acima de R$ 300 feitas na loja A:

```python
df = pd.DataFrame({
    'dia': ['seg', 'ter', 'qua', 'qui', 'sex'],
    'vendas': [250, 300, 280, 410, 380],
    'loja': ['A', 'A', 'B', 'B', 'A']

## üìé Dicas e Armadilhas

- Sempre use par√™nteses em express√µes com `&`, `|`, `~`.
- `.isin()` √© √∫til para m√∫ltiplos valores, mas cuidado com mai√∫sculas/min√∫sculas.
- `.between()` inclui os limites (ex: 10 e 50 est√£o dentro).
- Evite usar `==` com valores `NaN` ‚Äî use `isna()` ou `notna()`.
- Combine filtragem com `loc[]` para selecionar colunas espec√≠ficas.
